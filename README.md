# CryTex-P.AI

CryTex-P.AI is a web application for predicting crystallographic texture components from pole figure images using image processing and texture matching algorithms.

## Live Website

https://d22mt001-spec.github.io/CryTex-P.AI/

## Features

- Upload {100}, {110}, and {111} pole figure images
- Automatic spot detection
- MRD-based filtering
- Texture component identification
- Texture summary generation
- Downloadable results

## How to Use

1. Upload all three pole figure images.
2. Set MRD limits (min and max) and angle tolerance.
3. Click "Analyze All Three".
4. Review detected texture components.
